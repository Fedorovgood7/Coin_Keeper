# CoinKeeper Frontend

Приложение для учёта личных финансов на React + TypeScript.

##  Быстрый старт

### Установка зависимостей

```bash
cd src
npm install
```

### Запуск dev сервера

```bash
npm run dev
```

Приложение откроется на http://localhost:5173

### Сборка для production

```bash
npm run build
```

Результат в папке `dist/`

### Предпросмотр production сборки

```bash
npm run preview
```

## 📱 Функциональность

### Главная страница
- Общий баланс по всем счетам
- Круговая диаграмма расходов
- Список категорий с прогрессом
- Быстрое добавление операции

### Операции
- Список транзакций с группировкой по датам
- Фильтры по типу (расходы/доходы/переводы)
- Выбор периода (день/неделя/месяц/год)
- Фильтр по категориям

### Добавление операции
- Выбор типа (расход/доход/перевод)
- Сетка категорий с иконками
- Выбор счёта
- Дата и описание
- Регулярные операции

### Счета
- Список счетов с балансами
- Общий баланс
- Добавление/редактирование счетов
- Выбор типа (карта/наличные/накопительный)
- Выбор цвета

### Аналитика
- Статистика доходов/расходов
- График расходов по дням
- Динамика по месяцам
- Расходы по категориям

### Бюджет
- Месячный бюджет с прогрессом
- Безопасная сумма в день
- Лимиты по категориям
- Цели накоплений
- Пополнение целей

- Мобильные модалки

## 🛠 Технологический стек

- **React 18** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик
- **React Router** - маршрутизация
- **Zustand** - управление состоянием

## 📁 Структура проекта

```
src/
├── app/                    # Точка входа, стили
│   ├── App.tsx            # Главный компонент с роутингом
│   └── styles.css         # Глобальные стили
├── pages/                  # Страницы приложения
│   ├── login/             # Авторизация
│   ├── dashboard/         # Главная
│   ├── transactions/      # Операции
│   ├── accounts/          # Счета
│   ├── analytics/         # Аналитика
│   └── budget/            # Бюджет
├── shared/                # Переиспользуемый код
│   ├── store/             # Zustand store
│   ├── lib/               # Утилиты
│   └── ui/                # UI компоненты
├── entities/              # Типы данных
├── main.tsx               # Entry point
└── index.html             # HTML шаблон
```

##  Авторизация

- Демо-режим (локальное хранение данных)
- Интеграция с Yandex ID (требует backend)

## 💾 Хранение данных

Данные хранятся в localStorage браузера через Zustand persist middleware.

## 📊 MVP функции

- ✅ Бюджет с лимитами

### Docker
```dockerfile

WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=builder /app/dist /usr/share/nginx/html
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```
